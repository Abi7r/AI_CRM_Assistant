#  AI CRM Assistant

AI-powered Customer Relationship Management system with natural language query interface

## Features

- Natural language CRM queries using Groq AI (Llama 3.3)
- Real-time customer data retrieval and analytics
- Revenue tracking and lead prioritization
- Intuitive chat interface

## Tech Stack

**Frontend:** React, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**AI:** Groq (Llama 3.3 70B)

##  Sample Queries

- "Show all active customers"
- "Top 5 customers by revenue"
- "High priority leads"
- "Total revenue from active customers"
